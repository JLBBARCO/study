Erro de login

Para resolver, acesse o Git Bash, digite:
git config --global user.name "__________"
git config --global user.email "__________"

GitHub

Visual Studio Code (VS Code)

Ao fazer login com a conta do GitHub, o VS Code baixa automaticamente todas as extensões necessárias para a vinculação.

Atalhos

    Estes comandos serão executados no Terminal GitBash do VSCode com a pasta que deseja enviar aberta.

  • Repositório SEM COMMITs

    1. git init

    2. git add README.md

    3. git commit -m "first commit"

    4. git branch -M main

    5. git remote add origin 'link do repositório'

    6. git push -u origin main

  • Repositório funcional

    1. git add --all

    2. git commit -m "nome do commit"

    3. git remote add origin 'link do repositório'

    4. git branch -M main

    5. git push -u origin main

Windows

Se estiver em Windows pode executar winget install para o GIT, e winget install GitHub.GitHubDesktop para o GitHub Desktop no Windows PowerShell como Administrador.

Se quiser baixar tudo junto, basta adicionar ; entre os comandos.
