As linguagens de programação são principalmente C e C++ e utiliza-se um programa chamado Arduino IDE para exportar para a placa.

Download

Se quiser pode fazer download do Arduino IDEpelo site. Ou pode executar o Windows PowerShell como ADMINISTRADOR e executar o comando winget install ArduinoSA.IDE.stable. Ou ainda, pode baixar pela Microsoft Store pesquisando por Arduino IDE.
